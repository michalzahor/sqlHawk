    Welcome to SqlHawk

    This tool is aimed at developers and sys admins who have to create / maintain / deploy new or existing sql databases. The aim is to create repeatable script based deployment & enforcement of schema and data changes.

    With this tool you can

    • Generate html documentation of an existing database
    • Extract procedures / views / functions from an existing database to a filesystem
    • Force the procedures / views / functions of a database to match the version stored on a filesystem
    • Apply scripted updates, and track which updates have been applied and when they were applied
